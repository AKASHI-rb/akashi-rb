# 行動規範 (Code of Conduct)

AKASHI.rb は、参加者全員が歓迎され、安心して過ごせるコミュニティを目指しています。
私たちは、性別、性的指向、障害、外見、身体的特徴、人種、宗教にかかわらず、誰にとってもハラスメントのないコミュニティ体験を提供することに尽力します。

## 私たちの基準

より良いコミュニティ環境を作るために、以下の振る舞いを推奨します：

- 友好的で包摂的な言葉を使うこと
- 異なる視点や経験を尊重すること
- 建設的な批判を受け入れること
- コミュニティ全体にとって何が最善かを考えること
- 他の参加者に対して共感を示すこと

以下の振る舞いは容認しません：

- 性的な言葉や画像の使用、および不適切な性的関心
- トローリング、侮辱、個人攻撃
- 公的または私的な嫌がらせ
- 他人の個人情報を許可なく公開すること
- その他、職業的環境において不適切と見なされる行為

## 適用範囲

この行動規範は、イベント会場、オンラインスペース（Discordなど）、および AKASHI.rb を代表するあらゆる場に適用されます。

## 報告と執行

ハラスメントや不適切な行動を目撃した場合は、直ちにオーガナイザー（[無双](https://github.com/nishioka-shinji)、[さご](https://github.com/sago35)）に連絡してください。
オーガナイザーは、コミュニティの安全を守るために適切と判断される行動（警告、イベントからの退場処分など）をとる権利を有します。
